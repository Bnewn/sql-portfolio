# 8 Week SQL Challenge

This folder contains my personal solutions to the [8 Week SQL Challenge](https://8weeksqlchallenge.com/) created by Danny Ma.

## Introduction
The [8 Week SQL Challenge](https://8weeksqlchallenge.com/) features 8 different case studies, each presenting unique business scenarios and datasets. From analyzing customer dining patterns to optimizing digital marketing campaigns, these challenges cover essential SQL concepts used in real data analyst roles.

Each case study folder contains my approach, solutions, and key insights discovered through the data.
